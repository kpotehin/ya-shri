Задание №2: Опишите своими словами, что происходит в браузерах при клике на ссылку yandex.ru (полнота ответа – на ваше усмотрение).

Инструменты:
  1. Пользователь набирает домен в адресной строке или переходит* по ссылке (yandex.ru). В этот момент браузер разбивает введённый домен на имя хоста и путь к файлу.
  2. Если какая-то информация имеется в кэше, она берётся оттуда и сразу переходим к пункту 14, иначе идём по порядку.
  3. Запрос отправляется на DNS-сервера провайдера (если пользователь не прописал соответствия в файле hosts).
  4. Если в кэше DNS-серверов провайдера существуют IP-адрес, привязанный к искомому домену, то сразу переходим к пункту 14, иначе идём по порядку.
  5. DNS-сервер провайдера делает запрос к корневому DNS-серверу.
  6. Корневой DNS-сервер находит адреса DNS-серверов хостера, к которому привязаны NS-серверы искомого домена и возвращает их DNS-серверу провайдера.
  7. DNS-сервер провайдера делает запрос к одному из NS-серверов.
  8. NS-сервер возвращает IP-адрес искомого домена.
  9. DNS-сервер провайдера кэширует этот адрес и отдаёт его браузеру.
  10. Браузер открывает TCP/IP-соединение к серверу, на котором лежит ресурс, и отправляет HTTP-запрос.
  11. Браузер получает ответ от сервера, и может закрыть TCP/IP-соединение или сделать другой запрос.
  12. Браузер смотрит HTTP-заголовки, предварительно получив их из ответа, и если код состояния HTTP не соответствует шаблону 2xx, то выполняет различные действия в зависимости от кода состояния. В противном случае идёт дальше.
  13. Если HTTP-заголовки не запрещают кэширование данных, то они кэшируются браузером.
  14. Браузер декодирует ответ сервера.
  15. Браузер разбирает получившийся результат, преобразуя его в пригодную для чтения и выполнения структуру, формируется дерево DOM.
  16. Браузер выстраивает элементы для отображения в том порядке, как они будут выводиться на экран.
  17. Браузер отображает (рисует) получившийся результат.
* Если пользователь кликает по ссылке, то срабатывают различные javascript события (onMouseDown, onMouseUp, onClick, onDblClick - опционально, onBeforeUnload – в некоторых браузерах, onUnload), обработчики которых могут оборвать переход по ссылке.

function Car(manufacturer, model, year) {
    this.manufacturer = manufacturer;
    this.model = model;
    this.year = typeof year != 'undefined' ? year : new Date().getFullYear();

    this.getInfo = function() {
        return this.manufacturer + ' ' + this.model + ' ' + this.year;
    }

    this.getDetailedInfo = function() {
        return 'Производитель: ' + this.manufacturer + '. Модель: ' + this.model + '. Год: ' + this.year;
    }

    this.setPrice = function(price) {
    	this.price = price;
    }
}

Car.prototype.toString = function () {
	return this.getInfo();
};

var bmw = new Car('BMW', 'X5', 2010),
    audi = new Car('Audi', 'Q5', 2012),
    toyota = new Car('Toyota', 'Camry');

function CarDealer(name) {
    this.name = name;
    this.cars = [];

    this.add = function() {
    	for (var i=0; i < arguments.length; i++)
    		this.cars.push(arguments[i]);

		return this;
    }

    this.setPrice = function(carInfo, price) {
    	if (/^(€|¥)[0-9]/.test(price)) {
			this.cars.forEach(function (car) {
				if (car.getInfo() == carInfo)
					car.setPrice(price);
			});
		}

		return this;
    }

    this.list = function() {
    	var cars = [];

		this.cars.forEach(function (car) {
			cars.push(car.getInfo());
		});

		return cars.join(', ');
    }

    this.listByCountry = function(country) {
    	var cars = [];

		this.cars.forEach(function (car) {
			if (country == getCountry.call(car))
				cars.push(car.getInfo());
		});

		return cars.join(', ');
    }

    this.listWithPrice = function() {
    	var cars = [], price, currency, course;

		this.cars.forEach(function (car) {
			currency = car.price.replace(/[0-9]/g, '');
			price = car.price.replace(/[^0-9]/g, '');

			if (currency == '€')
				course = 40.7249;
			else
				course = 0.415156;

			price = price * course;

			cars.push(car.getInfo() + ' ' + Math.round(price) + ' руб.');
		});

		return cars.join(', ');
    }
}

Examples:
 printargs.sh a b c
 printargs.sh -v a b c
 printargs.sh -m 'Arguments count: ' a b c

Python 
#!/usr/bin/env python

import argparse

parser = argparse.ArgumentParser(description='Print the number of arguments.')
parser.add_argument('arguments', metavar='ARG', type=str, nargs='*', help='some arguments')
parser.add_argument('-m', dest='message', default='', help='custom message')
parser.add_argument('-v', dest='verbose', default='', help='verbose mode')

args = parser.parse_args()

count = 0

for a in args.arguments:
    if args.verbose != '':
    	print(a)
    count += 1

if args.message != '':
    print(args.message)

print(count)

Perl 
#!/usr/bin/env perl

use strict;
use Getopt::Long;
use Pod::Usage;

my $help;
my $message = '';
my $verbose;

@ARGV and GetOptions(
    "h" => \$help,
    "m:s" => \$message,
    "v" => \$verbose,
) or pod2usage(1);
pod2usage(-verbose => 2, -exitval => 2) if $help;

my $count = 0;
my $arg;

foreach $arg(@ARGV) {
    if ($verbose) {
        print $arg . "\n";
    }
    $count++;
}

if ($message ne "") {
    print($message . "\n");
}

print($count . "\n");

__END__
